npm init -y :Para empezar un proyecto de node 
npm i express : para instalar la libreria express 

const express = require("express");  //Se llama a exprees

const app = express();   //Se inicializa express y se guarda en app.

app.listen(3000, () => {                                        //Un ejemplo de hacer correr el servidor en el puerto 3000.
    console.log("Servidor escuchando en el puerto 3000");       //Un ejemplo de hacer correr el servidor en el puerto 3000.
});                                                             //Un ejemplo de hacer correr el servidor en el puerto 3000.

node app : para abrir el servidor y la terminal bash se va a bloquear eso significa que esta corriendo bien.

control + c : para detener el servidor

npx nodemon app : para iniciar el servidor y lo mas importante es que cada vez que se realiza un cambio
se reinicia el servidor automaticamente. 

// Se puede configurar el package.json en la parte de las script para que sea mas practico de la siguiente manera:
"start":"node app",
"dev":"npx nodemon app" 

para ejecutar el script (como desarrollador) en consola se pone : npm nodemon app 
para ejecutar el script (no entorno de desarrollo ) en consola se pone : node star
